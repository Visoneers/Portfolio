Pinaka.in is an interactive e-commerce solution providing users with an opportunity to buy fashion products according to their choice. In this website we have basically 2 modules. The first module includes the customer module and second module includes admin module. In customer module a customer can buy various fashion outfits and can proceed to payment via PayPal window. The admin module contains the access of admin page on the website. The admin can change everything in the website. He has the ability to add, delete, and update any information regarding the clothing products.
